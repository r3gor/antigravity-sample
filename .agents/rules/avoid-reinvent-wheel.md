# Rule: Avoid Reinventing the Wheel

- **Codebase First:** Before writing any utility function, custom state management, or complex algorithmic logic, ALWAYS search the existing codebase to see if it's already implemented.
- **Use Established Libraries:** If the functionality is not present in the codebase, strongly prefer recommending a well-established open-source library or built-in framework feature over writing custom code from scratch.
- **Custom Code as Last Resort:** Only write custom implementation from scratch when no standard solution exists, or when the user explicitly instructs you to do so.
