## Diference between rules and workflows in antigravity

Rules: This are mandatory instructions that the agent must always follow.
Example: 
- Dont use comments
- Dont use print statements


Workflows: This are optional instructions that the agent can follow if it deems it appropriate.

Example:
- Do analysis of security of the code
- Do analysis of performance of the code

Skills: This are the skills that the agent has to perform the task.
